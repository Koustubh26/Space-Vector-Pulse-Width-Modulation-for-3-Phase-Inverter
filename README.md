# Space-Vector-Pulse-Width-Modulation-for-3-Phase-Inverter

**Introduction**

Space Vector Pulse Width Modulation is a PWM technique used to produce switching signals to control an inverter. It is very popular for controlling inverters since it offers 15% increase in the DC link voltage utilization and low output harmonic distortions compared to the conventional sinusoidal PWM. Its output is similar to the output of a Third Harmonic injected inverter in terms of voltage output.
